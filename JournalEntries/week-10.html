Week 10

November 10 - November 14, 2025

This week we learned how to connect Flask with MongoDB. The in-class tutorial walked us through setting up a MongoDB Atlas cluster, configuring environment variables, and securely connecting the database to our Flask app. We practiced inserting, retrieving, and updating data from a collection, and explored how to use logical operators for filtering results. It was interesting to see how Flask can dynamically communicate with a live database in the cloud instead of relying on static files.

Reflections

I already had prior experience working with databases, including MongoDB, so this week's lesson felt more familiar and straightforward for me. Setting up the environment variables and connecting Flask to MongoDB using MONGO_URI was something I had done before, but this time I understood more clearly how Flask handles the connection through Flask-PyMongo. The hands-on exercises helped reinforce my understanding of CRUD operations in a web app context. This week also made me think about how I could apply these concepts to my current project. Right now, my Split Decision app with Cathy stores user data in JSON files, but i'm thinking about switching to a database instead in the near future, maybe for project 3 or for my other class project. Using MongoDB would make the data storage more scalable and secure, and it could help us better manage user interactions.
